# Mochy

Simple test framework inspired by Mocha

## Installation

Via npm on Node:

```
npm install mochy
```

or globally

```
npm install mochy -g
```

## Usage

```
mochy <name> [<name> ... ]
```
where `name` is a directory or a `.js` file. If `name` is a directory, it processes all the `.js` files in
directory.
